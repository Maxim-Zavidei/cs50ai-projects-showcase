    Degrees

    The problem consists of finding the shortest path between any two actors by choosing a sequence of movies that connects them.

    For example, the shortest path between Jennifer Lawrence and Tom Hanks is 2 movies: Jennifer Lawrence starred with Kevin Bacon in “X-Men: First Class,” and Kevin Bacon starred with Tom Hanks in “Apollo 13.”

    The final solution is a Python script that uses Breadth First Search Algorithm to accomplish this.
