    Degrees

    The problem consists of finding the shortest path between any two actors by choosing a sequence of movies that connects them.

    For example, the shortest path between Jennifer Lawrence and Tom Hanks is 2 movies: Jennifer Lawrence starred with Kevin Bacon in “X-Men: First Class,” and Kevin Bacon starred with Tom Hanks in “Apollo 13.”

    The final solution is a Python script that uses Breadth First Search Algorithm to accomplish this.

    Knights and Knaves Puzzle 

    Src  

    The following information is given in the this puzzle: Each character is either a knight or a knave. A knight will always tell the truth: if knight states a sentence, then that sentence is true. Conversely, a knave will always lie: if a knave states a sentence, then that sentence is false.

    Given a set of sentences spoken by each of the characters, write an app that determines, for each character, whether that character is a knight or a knave.
